Règles

  1. Il faut résoudre dans l'ordre successivement chacun des 5 problèmes.
  2. Les programmes doivent être réalisés en Java.
  3. Quand vous envoyez une solution, votre programme est soumis à un jeu de données de test pour vérifier qu'il fonctionne correctement. Il devra donc lire ces données d'entrée et générer des données de sortie.
  4. Pour lire les données d'entrée, utilisez un Scanner branché sur System.in.
  5. Pour écrire les données de sortie, utilisez System.out.print() ou System.out.println().
  6. La classe qui contient votre main doit être placée en fin de programme, et ne doit pas être déclarée public.
  7. Le statut de votre solution est alors visible sur le tableau de bord.
